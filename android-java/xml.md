# xml 사용하여 생산성높이기

| xml |  |
| :--- | :--- |
| colors.xml | 색상을 설정하고 이름을 지정해서 사용 |
| styles.xml | 스타일을 설정하고 이름을 지정해서 재사용 |
| dimens.xml | 길이를 설정하고 이름을 지정해서 재사용 |
| string.xml | 텍스트를 설정하고 이름을 지정해서 재사용 |

#### 사용법

android:text="@string/strSummary"
