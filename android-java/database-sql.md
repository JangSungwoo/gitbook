# DataBase, SQL

## DataBase 

데이터가 저장된 저장소. 

### 관계형 데이터 베이스 

**키와 값들의 간단한 관계를 테이블화 시킨 매우 간단한 원칙의 전산정보 데이터베이스.** 

실무에서는 MySQL, Oracle 과 같은 관계형 데이터 베이스를 많이 볼 수 있음 

### 데이터베이스 4단계 

1단계 Open 

2단계 Table 생성 

3단계 추가

4단계 조회
