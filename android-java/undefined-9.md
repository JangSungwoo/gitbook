# Permission

## 일반 권한과 위험권한\(마시멜로 API23부터\)

#### 일반 권한 : 설치 시 사용자의 설치 수락을 통해 사용자 승인을 받음 

#### 위험 권한 : 실행 시 사용자의 위험권한 허용을 통해 사용자 승인을 받음
